Заголовок: LearnTrack
Опис продукту: Цей продукт має забезпечити студентам простий і швидкий доступ до завдань без складних меню, враховуючи можливі технічні обмеження (слабкий інтернет чи старі пристрої). Викладачі отримають інструмент для швидкого додавання та перевірки завдань, що зменшить витрати часу на організацію навчального процесу. Адміністратори зможуть керувати групами та доступами, забезпечуючи порядок у системі. Таким чином, продукт сприятиме підвищенню дисципліни та якості навчання.

План розробки
Спершу створюється механізм реєстрації та входу, що дозволяє формувати особисті акаунти. Далі впроваджується можливість створення та перегляду курсів із темами й уроками, а також відстеження прогресу через позначення завершених матеріалів і відсоток виконання. Наступним етапом є додавання навчальних ресурсів у форматі текстів, відео та тестів, інтерактивне проходження завдань і формування базової статистики. Завершує план впровадження простого механізму нотифікацій для інформування про нові матеріали та дедлайни.

Поточний стан проєкту
Поточний статус виконання проекту LearnTrack можна охарактеризувати як етап активної реалізації основних MVP-функцій. Уже створено базову систему реєстрації та входу, а також прототип структури курсів із темами та уроками. Ведеться робота над механізмом відстеження прогресу та інтеграцією навчальних матеріалів, включно з тестами. Паралельно розробляється проста статистика та нотифікації, що дозволить завершити мінімально життєздатний продукт для перших користувачів.
